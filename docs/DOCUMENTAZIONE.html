1. Panoramica Generale

L'applicazione Telaio 3D + Nucleo in c.a. è uno strumento per l'analisi strutturale che consente di modellare edifici multipiano con telaio tridimensionale in cemento armato e nucleo irrigidente a pareti. Il nucleo viene modellato come mesh di elementi shell quadrilateri, mentre il telaio utilizza elementi beam tridimensionali con proprietà elastiche complete.

ⓘ Destinazione d'uso

Strumento destinato a ingegneri strutturali per la modellazione e analisi di strutture intelaiate con nuclei in calcestruzzo armato, sismoresistenti secondo le normative vigenti italiane ed europee.

6. Metodologia di Calcolo

6.1 Formulazione Elemento Beam 3D

Equazione constitutiva elemento elasticBeamColumn
{Flocal} = [Klocal] · {ulocal}

L'elemento elasticBeamColumn utilizza la formulazione di trave di Timoshenko con 6 gradi di libertà per nodo (3 traslazioni + 3 rotazioni). La matrice di rigidezza locale include i termini di rigidezza assiale, flessionale e torsionale.

Matrice di rigidezza assiale
Kax = E · A / L
Matrice di rigidezza flessionale (piano locale y-z)
Kflex = E · I / L · [12, -6L, -12, -6L]
Mi = (E·I/L) · (6/L·uj - 2·θi + 4·θj)

6.2 Formulazione Elemento Shell MITC4

L'elemento ShellMITC4 (Mixed Interpolation of Tensorial Components) è un elemento quadrilatero a 4 nodi con integrazione ridotta. Utilizza l'ipotesi di lastra di Mindlin-Reissner con gradi di libertà di membranatura e flessione disaccoppiati.

Sezione ElasticMembranePlateSection
[D] = (E·h³ / 12(1-ν²)) · [
  1    ν    0
  ν    1    0
  0    0  (1-ν)/2
]

dove:

  • E = modulo di Young (MPa)
  • ν = coefficiente di Poisson
  • h = spessore della parete (m)

6.3 Algoritmo di Analisi Statica Lineare

Sistema di equazioni
[K] · {U} = {F}

dove:

  • [K] = matrice di rigidezza globale della struttura
  • {U} = vettore degli spostamenti nodali incogniti
  • {F} = vettore dei carichi nodali equivalenti
ⓘ Parametri algoritmo di soluzione

L'implementazione utilizza:
• Sistema: BandGeneral (matrice a banda)
• Numberer: RCM (Reverse Cuthill-McKee)
• Constraints: Plain
• Test: NormDispIncr con tolleranza 10⁻¹²
• Algorithm: Linear
• Integrator: LoadControl con incremento unitario

6.4 Discretizzazione Carichi Distribuiti

Per elementi beam 3D, i carichi distribuiti variabili (trapezoidali) vengono discretizzati in segmenti uniformi equivalenti:

Carico equivalente per segmento k (k = 1...n)
qk = q0 + (q1 - q0) · (k-0.5) / n

dove n è il numero di segmenti (parametro configurabile, default = 10).

8. Esempio di Utilizzo

8.1 Workflow Tipico

  1. Avvio applicazione: streamlit run app.py
  2. Configurazione geometria: Impostare dimensioni telaio e nucleo nel pannello laterale
  3. Generazione modello: Cliccare "Genera Nucleo" per creare la mesh automatica
  4. Modifica parametri: Utilizzare i tab per modificare nodi, elementi, carichi
  5. Validazione: Cliccare "Valida modello" per verificare coerenza input
  6. Esecuzione analisi: Cliccare "Solve ▸ Linear Static"
